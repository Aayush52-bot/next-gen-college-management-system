# Next-Gen College Management System

A comprehensive web-based College Management System developed to digitize and streamline academic and administrative processes. The platform provides separate dashboards for students and administrators, enabling efficient management of admissions, student records, courses, fees, attendance, and academic information through a centralized system.

---

## Features

- **Online Admission and Registration System**: Students can apply and register online without paperwork.
- **Student Login and Dashboard**: Personalized dashboard for each student to access their academic information.
- **Course and Academic Information Management**: Manage courses, schedules, and academic details efficiently.
- **Fee Management and Payment Integration**: Track and manage student fees with payment support.
- **Student Record and Profile Management**: Centralized storage of all student data and profiles.
- **Attendance and Result Tracking**: Monitor attendance records and academic results in real time.
- **Admin Panel for Complete System Control**: Full control over the system for administrators.
- **Responsive and User-Friendly Interface**: Works seamlessly on desktops, tablets, and mobile devices.
- **Secure Authentication and Data Management**: Role-based login with secure data handling.

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Aayush52-bot/college-management-system.git
   ```

2. Move the project folder to your XAMPP `htdocs` directory:
   ```
   D:/xampp/htdocs/college-management-system/
   ```

3. Start **Apache** and **MySQL** in XAMPP Control Panel.

4. Import the database:
   - Open `http://localhost/phpmyadmin`
   - Create a new database named `college_db`
   - Import the provided `.sql` file from the `/database` folder

5. Configure database connection:
   - Open `db.php`
   - Update your database credentials if needed

6. Open in browser:
   ```
   http://localhost/college-management-system/
   ```
